# Games-Application

This is a fun and engaging Windows Forms application built with C# that offers a collection of classic games, including a modern twist on Hangman ("Spanzuratoarea") and a challenging Puzzle game. Players can log in to track their scores and progress, with flexible login options including username-only or username and password.


# Features
- Two Exciting Games:
  - Spanzuratoarea (Hangman): Guess the word letter by letter before the flower withers away! Earn points based on your accuracy.
  - Puzzle Game: Challenge your mind with a classic tile-sliding puzzle. (Assuming this is the other game based on "puzzle games")
- Flexible User Authentication:
  - Guest Mode: Play instantly without needing to log in.
  - User Login: Create an account to save your progress and scores. Supports both username-only and username + password authentication for added security.
 
# How to play
### Login Screen
- Enter a username to play. If it's a new username, an account will be created.
- Optionally, enter a password for existing users with password protection.

### Spanzuratoarea (Hangman)
- Click "Start" to begin a new round.
- Guess letters by clicking on the alphabet buttons.
- Correct guesses reveal letters in the word. Incorrect guesses affect the flower's state.
- Win by guessing the word, or lose if the flower fully withers.
